# Simple Diagram APP

A lightweight browser-based diagram and flowchart tool built using React .

## Features
- Add, remove, and rename nodes
- Resizable nodes with drag handles
- Connect nodes with arrows
- Upload images as resizable nodes
- Save/load diagrams to JSON file or browser storage

## Stack
- React
- ReactFlow
- Azure Blob Storage *(coming soon)*

## Run locally
```bash
npm install
npm start
```

## Deploy
```bash
npm run build
./deploy.sh
```
